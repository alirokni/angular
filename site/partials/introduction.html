Rokni.net is an 'experimental' site; some sort of a 'sandbox'. I try new technologies here and then might move some of it to Rokni.com.

Anything related to html, css, javascript, design patterns and frameworks that is interesting and needs to be tried out, will be tested here.

An interesting topic that I am following and exploring as is, Web Performance. I am testing, analyzing and optimizing this site and the Rokni.com with Google PageSpeed Tools, YSlow and some other interesting online tools to make the site faster.

Currently, none of the scripts and stylesheets are compressed and or minified, eventually, those files should be minified and concatnated either using a task runner (similar process to my github node-mustache project -rokni.net:8080/index.html-) or Closure Compiler Service UI can be used.

As this site considered Hard Hat Area, best to view it with modern browsers. So, older browsers are not supported and might not rendered correctly.
